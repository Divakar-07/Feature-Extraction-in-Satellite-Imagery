Our Methodology

1. Satellite Image Acquisition

We collect high-resolution satellite imagery from public and commercial sources such as Sentinel-2, Landsat, and SpaceNet datasets. Multispectral images are prioritized to allow for accurate classification of land cover features.

2. Preprocessing

The raw satellite data undergoes preprocessing which includes radiometric correction, cloud masking, geometric alignment, and resizing. This ensures consistency across temporal data for accurate model training.

We also enhance contrast and normalize bands to prepare for deep learning segmentation.

3. Data Augmentation

To address class imbalance and enrich the dataset, we apply augmentation techniques like rotation, flipping, brightness changes, and simulated terrain noise. This step significantly boosts model generalization.

4. Deep Learning-Based Feature Extraction

We use U-Net as the base architecture for semantic segmentation. To enhance performance on limited or domain-shifted data, we use CycleGANs for synthetic image generation that better represent real-world variability.

Classes include water, roads, vegetation, buildings, unpaved land, and miscellaneous.

5. Evaluation and Post-Processing

Model outputs are evaluated using metrics like Mean Intersection over Union (mIoU), precision, and recall. Post-processing includes morphological operations to reduce noise and improve region consistency.

In our test, U-Net+CycleGAN achieved a mIoU improvement of 5% over baseline U-Net.

6. Visualization & Export

The extracted features are visualized over satellite imagery using transparent overlays. Final outputs are exported in formats compatible with GIS tools (GeoTIFF, SHP), and also rendered in interactive web dashboards for stakeholder access.
